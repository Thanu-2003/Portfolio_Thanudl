SQL

SQL is used to communicate with and manage data in relational databases. DML: Includes commands like SELECT, INSERT, UPDATE, and DELETE to manage data. DDL: Commands like CREATE, ALTER, DROP are used to define database structures such as tables, indexes, and constraints. Joins: Used to combine data from multiple tables based on common columns (INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL JOIN). Constraints: Rules like PRIMARY KEY, UNIQUE, NOT NULL, and FOREIGN KEY ensure data integrity.

HTML

Basic Structure: Every HTML document starts with !DOCTYPE html, followed by html, head, and body tags. Elements and Tags: Use tags like p, h1 to h6, a, img to create and structure content. Links: Use the a href="URL" tag to create hyperlinks for navigation between pages. Forms: Collect user input with form elements and input types like input, textarea, and button. Media Support: Embed images (img), videos (video), and audio (audio) directly in web pages.

CSS

Styling: CSS is used to style HTML elements, such as setting colors, fonts, margins, and layouts. Selectors: CSS uses selectors to target HTML elements, such as: Tag selectors (e.g., h1, p) Class selectors (e.g., .class-name) ID selectors (e.g., #id-name). Box Model: Understand the box model consisting of margin, border, padding, and content to control spacing and element sizes. Flexbox/Grid: Layout systems for responsive and flexible designs. Responsive Design: Use media queries to create designs for different screen sizes.

DATA STRUCTURES

Arrays: Contiguous memory storage for elements, accessed via indices. Linked Lists: A sequence of nodes where each node points to the next one in the list. Stacks: LIFO (Last In, First Out) structure for storing elements. Queues: FIFO (First In, First Out) structure for managing elements. Trees: Hierarchical data structure with nodes and edges, including binary trees, AVL trees, etc.
